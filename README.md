# PROF_MARSHAL
VR training system for airport ground operations with real-time hand signal recognition &amp; animated aircraft feedback. Features Training, Practice &amp; Assessment modes with motion-capture-based pose matching, 10-min timed tests, 3-attempt limits &amp; score tracking. Built with Unity, C#, OpenXR &amp; Meta Quest 2. ISO/IEC 25010 validated with 57 participants
